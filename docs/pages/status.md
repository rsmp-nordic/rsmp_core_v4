## Status
```
<node>/status/<code>[/<component>]
```

Examples:
```
dk/cph/45fe/status/tlc.14        # S0014 current signal plan for the main component on node 45fe
dk/cph/45fe/status/tlc.25/sg/1   # S0025 time-of-green for signal group 1 on node 45fe
```
