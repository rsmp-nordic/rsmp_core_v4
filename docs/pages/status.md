## Status
```
<node>/status/<code>[/<component>]
```

Examples:
```
dk/cph/45fe/status/tlc.plan      # current signal plan for the main component on node 45fe
dk/cph/45fe/status/tlc.plans     # available plans for the main component on node 45fe
dk/cph/45fe/status/tlc.tog/sg/1  # S0025 time-of-green for signal group 1 on node 45fe
```
