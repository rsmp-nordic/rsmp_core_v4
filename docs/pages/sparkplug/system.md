## System Metrics

Metrics related to system functionality.

| Metric                 | Description          |
|------------------------|----------------------|
| **/sys/version**       | Version information  |
| **/sys/clock**         | Current system time  |

### Version information
Version information is in **/sys/version**.

### Clock
The metric **/sys/clock** indicates the current system time, which is essential for coordinating time-based operations, logging events, and ensuring overall system consistency and reliability.
