# Traffic Situations

| Metric                               | Description                  |
|--------------------------------------|------------------------------|
| **tlc/tc/1/traffic_situation/list**  | Available traffic situations |
| **tlc/tc/1/traffic_situation**       | Current traffic situation    |

## Usage
The metric `tlc/tc/1/traffic_situation/available` list all configured traffic situations.

The metric `tlc/tc/1/traffic_situation` indicates the currently active traffic situation, and associated metadata like the source of the current activation.
Change the metric to set or clear the current traffic situation.
