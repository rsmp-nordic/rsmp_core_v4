## Control Mode Metrics

Metrics for managing configuration.

| Metric                      | Description                  |
|-----------------------------|------------------------------|
| **/tlc/config/checksum**    | Config checksum    |
| **/tlc/config/download**    | Controller is switched on    |

### Configuration
The metric **/tlc/config/checksum** provides a checksum of all configations, to make it easy to verify that the correct set of configurations are used.s
The metric **/tlc/config/download** can be used to download a compressed binary file containing all necessary configurations.
