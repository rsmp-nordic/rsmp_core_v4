# Alarm
```
<node>/alarm/<code>[/<component>]
```

Examples:
```
45fe/alarm/tlc.hardware.error   # serious hardware error on node 45fe
45fe/alarm/tlc.detector/dl/7    # detector error for component dl/7 on node 45fe
```
